Функции | Список изменений | Горячие клавиши

Список горячих клавиш программы «Архив документов»

Основное окно

F1 Справка (список изменений и список горячих клавиш).
F3 Поиск документов.
F4 Изменение порядка группировки в дереве папок.
Alt + F4 Сворачивание программы. То же самое происходит при нажатии на крестик в правом верхнем углу основного окна программы. Обратите внимание, что при этом программа не закрывается, а сворачивается в «трей» — правый нижний угол экрана. Чтобы закрыть программу, нужно выбрать пункт «Закрыть программу» в контекстном меню, которое появляется при щелчке правой кнопкой мыши на иконке программы в трее.
F5 Обновление дерева папок и списка документов.
F10 Переход в основное меню.
Ctrl + F Настройки фильтра отображаемых документов. Этот фильтр позволяет ускорить работу, но если вдруг вы не найдете какой-то документ, который раньше вам был виден — проверьте настройки фильтра. Возможно, документ не виден из-за фильтра.
Ctrl + M Настройки сообщений и подтверждений.

Документ

F2 Сохранение документа. Для документов вне архива — сохранение в архив. Для документов в архиве — сохранение изменений, внесенных в изображение документа.
Ctrl + F2 Сохранение части документа в архив.
F7 Отправка сообщения по документу в архиве.
Ctrl + W Открытие документа в новом (вторичном) окне.
Ctrl + S Сканирование нового документа.
Ctrl + Del Удаление части документа (со сканера).
Ctrl + B Переход на предыдущую страницу изображения.
Ctrl + N Переход на следующую страницу изображения.
Ctrl + R Поворот страницы изображения вправо (по часовой стрелке).
Ctrl + L Поворот страницы влево (против часовой стрелки).
Ctrl + P Печать текущей страницы документа. Весь документ печатается только через меню!
Пробел Отметка сообщений по документу в архиве как прочтенные.
Del Удаление документа из работы (для документов в работе).

Вторичные окна

Alt + F4 Закрытие окна. В отличие от основного, вторичные окна закрываются, а не сворачиваются в трей.
